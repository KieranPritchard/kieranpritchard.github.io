BTEC First Foundation in Tech

Date Published: 21/02/2025

Topics: Education

A complete overview of my time on Bournemouth & Poole colleges, "T-Level Foundation in Digital" course. This page takes a brief look at each of the different units these are: online world, technology systems, web design and portfolio devlopement, digital graphics, and networking. In the future I am planning on expanding this page to have sub pages on each topic and assignment.

Starting out

Our first two months of this assignment was based on watching presentaitions on how networks work and making our own cables. While i cant show you anything related to the powerpoints we watched i can show you images from the network cables that we made. As according to our teacher, it is an essentail skill to have in indrustry. It took me a few attempts to get it right but once i did i had a functioning network cable.

Assignment one: The Features and Uses of Computer Networking

For this first assignment we had to make presentaition explaining the features of two seperate networks. To complete the task we had to explain two different uses and features of two types of network. For example, why is a LAN used and what is the purpose of PAN. then we had to review how they affect the user expierence and disscuss their strengths and weeknesses.

Assignment two: Create the Design

For this assignment we were tasked with creating a design for a hypohetical network. This involved describing the client requirements and purpose; listing the costs and the parts; making a of the network diagram; test plan and data; making alternative solutions and justifing my choices. This was a very different second assignment from the others in the way that it wasnt building ontop of them in the same way.

Assignment three: Develop and Test a Computer Network

This assignment then had us creating a network using virtual machines with virtual box. as part of this task we were premade windows 10 and windows server 2019 virtual machines which we used to complete the tasks. the tasks we were given were: to make three network users; make five shared folders and files, give the network a shared hardware and software resource. gather feedback and give suggustions on how i would improve it.

Assignment four: Review The Network

for the final assignment. The final task we had to complete involved reviewing the network againist the user requirements; how the feedback allows us to meet it; and then recommendations for improvement. Because of this task the overall assignment was very short and more like a reflection on how the unit went and the evolution of the network.
